## Getting Started

To run the program locally:

### Installation

1. Clone the repo and navigate to the local repository
   ```sh
   git clone https://github.com/394-w24/Tenny.git
   cd ../aStarAlgoGame
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run!
   ```sh
   npm run start
   ```
